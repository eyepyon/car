# Product Overview

ナンバープレート連動ウォレットシステム - 車両のナンバープレートをウォレットIDとして活用し、ブロックチェーン（Base L2）とAI（Qwen）を融合させた次世代モビリティプラットフォーム。

## Core Capabilities

1. **ナンバープレート認識** - AIカメラによる高精度なナンバープレート認識（Qwen-VL）
2. **ZKウォレット変換** - ゼロ知識証明によるプライバシー保護付きウォレットアドレス生成
3. **自動決済** - ゲート・ドライブスルーでのシームレスな自動課金
4. **P2P投げ銭** - 車両間での感謝の気持ちを伝えるチップ機能
5. **車両価値トークン化** - 車両査定額に基づく担保ローン機能

## Target Use Cases

- **駐車場自動決済**: AIカメラでナンバー認識 → 0.5秒でゲート解錠・決済
- **ドライブスルー**: 事前注文 → 到着時自動認識 → 待ち時間ゼロで受け取り
- **P2P投げ銭**: 道を譲ってくれた車へのお礼をブロックチェーンで送金
- **車担保ローン**: 車両価値を担保に即座に利用可能な信用枠を提供

## Value Proposition

- **フリクションレス**: 降車回数90%削減、窓開け操作不要
- **プライバシー保護**: ゼロ知識証明でナンバープレート情報を秘匿化
- **低コスト**: Base L2で$0.01以下のガス代、初期費用1/10（従来比）
- **自動運転時代対応**: 車内生活OSの基盤として将来の拡張性を確保
