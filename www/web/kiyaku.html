第1条：総則

1.1 規約の適用

本利用規約（以下「本規約」）は、株式会社 inCar（以下「当社」）が提供する inCar サービス（以下「本サービス」）の利用条件を定めるものです。ユーザーは本サービスを利用することにより、本規約に同意したものとみなされます。

1.2 定義

本規約において使用する用語の定義は以下のとおりです：

  • 「本サービス」：当社が提供する車両ナンバー認識による決済サービス、投げ銭機能、トークンローン機能等の総称
  • 「ユーザー」：本サービスを利用する個人または法人
  • 「車両情報」：車両のナンバープレート、車種、色等の識別情報
  • 「ウォレット」：本サービス内での決済に使用するデジタル財布

1.3 規約の変更

当社は、必要に応じて本規約を変更することができます。変更後の規約は、当社ウェブサイトまたはアプリケーション内での公表により効力を生じるものとします。

第2条：サービス内容

2.1 自動決済機能

本サービスは、車両のナンバープレートを認識することにより、駐車場料金、ドライブスルー料金等の自動決済機能を提供します。決済は事前に登録された支払い方法により実行されます。

2.2 投げ銭機能

ユーザーは他の車両のナンバープレートを撮影することにより、感謝の気持ちを金銭的に表現する投げ銭機能を利用できます。投げ銭の受取には本サービスへの登録が必要です。

2.3 トークンローン機能

当社は、ユーザーの車両査定額を基準として、一定額のトークンを貸与するサービスを提供します。貸与条件、利率、返済方法等は別途定める契約書に従います。

2.4 サービス提供時間

本サービスは原則として24時間365日提供されますが、システムメンテナンス、障害対応等により一時的に停止する場合があります。

第3条：ユーザーの権利と義務

3.1 ユーザーの権利

  • 本規約に従って本サービスを利用する権利
  • 個人情報の取り扱いについて説明を受ける権利
  • サービス内容について問い合わせを行う権利
  • 適切な手続きによりサービスを解約する権利

3.2 ユーザーの義務

  • 正確な情報を登録し、変更があった場合は速やかに更新すること
  • アカウント情報を適切に管理し、第三者に使用させないこと
  • 法令および本規約を遵守してサービスを利用すること
  • 他のユーザーや第三者の権利を侵害しないこと

3.3 禁止行為

ユーザーは以下の行為を行ってはなりません：

  • 虚偽の情報を登録または提供すること
  • 他人の車両情報を不正に使用すること
  • システムに不正アクセスまたは攻撃を行うこと
  • 本サービスを商用目的で無断利用すること
  • その他、当社が不適切と判断する行為

第4条：プライバシー保護

4.1 個人情報の収集

当社は、本サービスの提供に必要な範囲で、以下の個人情報を収集します：

  • 氏名、住所、電話番号、メールアドレス等の基本情報
  • 車両のナンバープレート、車種、色等の車両情報
  • 決済に関する情報（クレジットカード情報等）
  • 位置情報、利用履歴等のサービス利用情報

4.2 個人情報の利用目的

収集した個人情報は以下の目的で利用します：

  • 本サービスの提供および運営
  • ユーザー認証および本人確認
  • 決済処理および料金請求
  • カスタマーサポートの提供
  • サービス改善のための分析

4.3 データ保護技術

当社は、ユーザーの個人情報を保護するため、エンドツーエンド暗号化、ゼロ知識証明等の最新技術を採用し、データの安全性を確保します。

4.4 第三者への提供

当社は、法令に基づく場合または ユーザーの同意がある場合を除き、個人情報を第三者に提供しません。

第5条：支払い条件

5.1 決済方法

本サービスでは以下の決済方法をご利用いただけます：

  • クレジットカード（Visa、Mastercard、JCB、American Express）
  • デビットカード
  • 電子マネー（交通系 IC カード等）
  • 銀行振込
  • その他当社が指定する方法

5.2 料金体系

本サービスの利用料金は以下のとおりです：

  • 基本利用料：月額980円（税込）
  • 決済手数料：取引金額の2.5%
  • 投げ銭手数料：送金額の5%
  • トークンローン利息：年利15%

5.3 返金ポリシー

システム障害による決済エラー等、当社の責任による場合を除き、原則として返金は行いません。ただし、誤決済等の場合は、当社の判断により返金を行う場合があります。

5.4 トークンローン条件

トークンローンの詳細条件は以下のとおりです：

  • 貸付上限額：車両査定額の70%
  • 返済期間：最長36ヶ月
  • 遅延損害金：年利20%
  • 早期返済手数料：なし

第6条：サービス変更・終了

6.1 サービス内容の変更

当社は、サービス向上のため、事前の通知により本サービスの内容を変更することができます。重要な変更については、30日前までにユーザーに通知します。

6.2 サービスの一時停止

当社は、以下の場合にサービスを一時停止することができます：

  • システムメンテナンスまたはアップデート
  • 技術的障害の発生
  • 法令の変更への対応
  • その他運営上必要と判断される場合

6.3 サービスの終了

当社は、事業上の判断により本サービスを終了することができます。サービス終了の場合は、6ヶ月前までにユーザーに通知し、データの移行または削除について適切な措置を講じます。

6.4 ユーザーによる解約

ユーザーは、アプリケーション内の設定またはカスタマーサポートへの連絡により、いつでもサービスを解約することができます。解約時の残高は、ユーザーの指定する方法で返金します。

第7条：免責事項

7.1 サービス提供の免責

当社は、以下の事項について責任を負いません：

  • 天災、戦争、暴動等の不可抗力による損害
  • ユーザーの機器やネットワーク環境による障害
  • 第三者による不正アクセスやサイバー攻撃
  • ユーザーの規約違反による損害

7.2 技術的制約

車両ナンバー認識技術の特性上、以下の状況では正常に動作しない場合があります：

  • 悪天候（大雨、雪、霧等）による視界不良
  • ナンバープレートの汚れや損傷
  • カメラの故障や設置不良
  • その他技術的要因による認識エラー

7.3 損害賠償の制限

当社の責任による損害が発生した場合でも、当社の賠償責任は、ユーザーが当社に支払った直近12ヶ月間の利用料金を上限とします。

7.4 投資・金融リスク

トークンローン機能の利用に関して、金利変動、経済情勢の変化等による損失について、当社は責任を負いません。

8.1 準拠法

本規約は日本法に準拠し、日本法に従って解釈されるものとします。

8.2 管轄裁判所

本サービスに関する一切の紛争については、東京地方裁判所を第一審の専属的合意管轄裁判所とします。

8.3 分離可能性

本規約の一部が無効または執行不能と判断された場合でも、その他の条項の有効性には影響しないものとします。
